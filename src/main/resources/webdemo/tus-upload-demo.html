Resumable upload demo (TUS)

Pick a file. Try Stop mid-upload, then re-select the same file (or refresh the page and re-select it) to see it resume from the last byte the server has.

Pause: keeps the same upload alive in memory - press again to resume.
Stop: drops the upload object but keeps the URL in localStorage. Re-select the same file (even after a full page refresh) and it picks up where it left off.
Cancel: sends DELETE /api/tus/{id} (TUS Termination extension), removes the partial file from the server, and clears localStorage.

FileSizeState

Files on server: live view of /tmp/tus-uploads/, fetched from GET /api/tus. Auto-refreshes after upload / cancel / purge.
Purge: sends DELETE /api/tus to wipe every file and sidecar; also clears this browser's localStorage.
