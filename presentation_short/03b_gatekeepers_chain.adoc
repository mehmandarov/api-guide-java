=== Ch1 – The Filter Chain
==== Each Layer Is a Standard Jakarta EE API

---

```
Client Request
   │
   ├─ InputSanitizationFilter      ← @PreMatching – cleans query params
   ├─ InputSanitizationInterceptor ← ReaderInterceptor – sanitizes the body
   ├─ AuditFilter                  ← @NameBinding (@Audited) – opt-in
   │
   ▼
Resource(@Valid Session s)         ← @NotBlank, @Size, @NoProfanity
   │ fails?
   ▼
ConstraintViolationExceptionMapper → 400 + RFC 9457 JSON
```

_Three mechanisms, zero manual wiring – no framework magic._
