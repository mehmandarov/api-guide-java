=== Foundation — API Model ≠ Domain Model

---

```
Client ──[JSON]──► SessionDtoV1   ─┐
                   SessionDtoV2    ├─[.from()]──► Session ──► Repository
                   (future V3...) ─┘
```

* `domain/` — internal model, shaped for business rules
* `dto/` — API contract, shaped for clients, versioned independently
* `.from()` — the *Anti-Corruption Layer* between the two

_Rename a field internally → zero client impact._
