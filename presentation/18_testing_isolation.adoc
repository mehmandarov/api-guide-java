==== Test Isolation vs. Startup Time

* *Singleton container* — starts once (~3 s), shared by all 34 IT tests
* Trade-off: tests share mutable state (acceptable for demo with seed data)
* For full isolation: per-class container (~3 s × 6 classes instead of ~3 s total)

==== Running

```bash
# Unit only (no Docker)
mvn test -Pquarkus

# Full suite (requires Docker)
mvn verify -Pquarkus
```
