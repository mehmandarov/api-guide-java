=== Ch3 — Liveness Check
==== MicroProfile Health — Is the Process Alive?

---

link:../src/main/java/com/mehmandarov/confapi/observability/StartupHealthCheck.java[StartupHealthCheck.java]

[source,java]
----
@Liveness
@ApplicationScoped
public class StartupHealthCheck implements HealthCheck {

    @Override
    public HealthCheckResponse call() {
        return HealthCheckResponse.named("confapi-live")
                .withData("javaVersion", Runtime.version().toString())
                .up()
                .build();
    }
}
----

* `@Liveness` — Kubernetes uses this to decide whether to *restart* the pod
* Simple: if the process is alive, report UP
