=== Ch1 — The Filter Chain
==== How Gatekeepers Work

---

```
Client Request
  │
  ▼
┌────────────────────────────────┐
│ InputSanitizationFilter        │  ← @PreMatching — runs before URI matching
├────────────────────────────────┤
│ InputSanitizationInterceptor   │  ← ReaderInterceptor — sanitizes body
├────────────────────────────────┤
│ AuditFilter                    │  ← @NameBinding — activated on @Audited endpoints
├────────────────────────────────┤
│ Bean Validation (@Valid)       │  ← @NotBlank, @Size, @Min, @NoProfanity
└────────────────────────────────┘
  │
  ▼
Resource Method (clean input only!)
```

_Each layer is a standard Jakarta EE API — no framework magic._
